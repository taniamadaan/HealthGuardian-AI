#  Day 05 – Data Visualization using Matplotlib & Seaborn

**Date:** 6 July 2026

---

##  Objective

The objective of today's training session was to understand the importance of **Data Visualization** in Exploratory Data Analysis (EDA). The session focused on learning various visualization techniques used to represent data graphically, making it easier to understand distributions, patterns, trends, relationships, and correlations within a dataset.

---

##  Session Summary

Today's session introduced **Data Visualization**, one of the most important stages of Exploratory Data Analysis (EDA). Before building any Machine Learning model, it is essential to visualize the data to better understand its characteristics and uncover meaningful insights.

The trainer explained the significance of data visualization and demonstrated how graphical representations simplify complex datasets. Different visualization techniques were introduced using **Matplotlib** and **Seaborn**, two of the most widely used Python libraries for creating statistical and analytical graphs.

Throughout the session, various charts were discussed in detail, explaining when each visualization should be used, what information it provides, and how it helps during data analysis. The session also covered concepts such as data distribution, probability density, categorical analysis, correlation, and relationships between variables through different visualization techniques.

---

##  Topics Covered

### Introduction to Data Visualization

Data Visualization is the graphical representation of data that transforms raw numerical and categorical information into meaningful visual forms. It helps analysts identify trends, distributions, relationships, and anomalies more efficiently than observing raw tables.

---

### Matplotlib and Seaborn

The trainer introduced the two primary visualization libraries used in Python.

**Matplotlib** provides extensive customization for creating different types of graphs, whereas **Seaborn**, built on top of Matplotlib, offers attractive statistical visualizations with simplified syntax and better default styling.

---

### Histogram

A Histogram is used to visualize the frequency distribution of numerical data by dividing values into intervals called bins. It helps understand the spread of data, identify skewness, observe concentration of values, and detect unusual observations.

---

### Kernel Density Estimation (KDE)

Kernel Density Estimation (KDE) is a smooth representation of the probability distribution of numerical data. Unlike histograms, KDE plots provide a continuous density curve, making it easier to understand where observations are concentrated.

---

### Box Plot

A Box Plot summarizes numerical data using the five-number summary:

- Minimum
- First Quartile (Q1)
- Median
- Third Quartile (Q3)
- Maximum

It is widely used to analyze data spread, variability, and identify outliers.

---

### Count Plot

A Count Plot displays the frequency of each category within a categorical variable. It provides an easy way to compare the number of observations belonging to different categories.

---

### Bar Plot

A Bar Plot compares aggregated numerical values across different categories. It is commonly used to compare averages, totals, or other statistical measures between groups.

---

### Pie Chart

A Pie Chart represents categorical data as proportions of a whole. It is useful for understanding percentage distribution and comparing the contribution of different categories.

---

### Line Plot

A Line Plot visualizes continuous trends by connecting data points with lines. It is commonly used to observe changes and progression across ordered values.

---

### Violin Plot

A Violin Plot combines the characteristics of a Box Plot and a Kernel Density Plot. It displays both the statistical summary and the probability distribution of data, making it useful for comparing distributions across multiple categories.

---

### Scatter Plot

A Scatter Plot illustrates the relationship between two numerical variables by representing each observation as a point on a graph. It helps identify positive, negative, or weak relationships between variables as well as detect clusters and outliers.

---

### Correlation

Correlation measures the strength and direction of the relationship between two numerical variables. Correlation values range from **-1 to +1**, where positive values indicate a direct relationship, negative values indicate an inverse relationship, and values near zero indicate little or no correlation.

---

### Heatmap

A Heatmap is a color-coded representation of the correlation matrix. It provides an intuitive way to identify strongly correlated, weakly correlated, and negatively correlated variables within a dataset.

---

### Pair Plot

A Pair Plot generates multiple scatter plots and distribution plots simultaneously for selected numerical variables. It helps analyze relationships, distributions, and patterns among multiple features in a single visualization.

---

### Area Chart

An Area Chart is an extension of a Line Plot in which the area below the line is filled with color. It is useful for visualizing cumulative trends and emphasizing changes across ordered data.
