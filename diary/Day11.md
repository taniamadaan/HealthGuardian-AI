# Day 11 – Introduction to Supervised Learning & Logistic Regression

**Date:** 14 July 2026

---

## Objective

To understand the fundamentals of supervised machine learning, differentiate between classification and regression problems, and implement Logistic Regression for prediction using real-world datasets.

---

## Session Summary

Today's session introduced the concept of supervised learning, where machine learning models are trained using labelled data to learn patterns and make predictions on unseen data. The trainer explained the complete machine learning workflow, including data collection, feature engineering, train-test splitting, model training, evaluation, and prediction.

We also learned the difference between classification and regression problems with practical examples. Logistic Regression was implemented on a student placement dataset to predict placement status and later applied to a mental health dataset for depression prediction. The session also demonstrated how trained models can make predictions on new input data.

---

## Topics Covered

- Introduction to Supervised Learning
- Classification vs Regression
- Machine Learning Workflow
- Train-Test Split
- Logistic Regression
- Model Training and Prediction
- Accuracy Evaluation
- Prediction on New User Input

---

## Key Learning

I learned how supervised learning models use labelled data to identify patterns and make predictions. I also understood when to use classification and regression algorithms and gained practical experience implementing Logistic Regression for binary classification problems.

---

## Reflection

Today's session strengthened my understanding of the complete machine learning pipeline. Implementing Logistic Regression on multiple datasets helped me understand how a trained model can predict outcomes for unseen data and evaluate its performance using accuracy. These concepts will be directly useful while developing my Diabetes Detection project.
